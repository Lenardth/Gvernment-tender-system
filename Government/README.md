# Government Tender Monitoring System

## Overview

This project aims to create a transparent, efficient, and accountable tendering process for government projects. It provides a monitoring and verification system to ensure proper allocation and tracking of tenders.

## Features

- Register as a contractor
- Browse and apply for active tenders
- Investor & government dashboard for monitoring
- Tender cost calculator
- Verification and project tracking
